Правильно настроенное окружение разработки может сэкономить часы времени.

## Терминал

Я использую **iTerm2** на macOS с кастомной темой и плагинами zsh:

- `zsh-autosuggestions` — автодополнение команд
- `zsh-syntax-highlighting` — подсветка синтаксиса
- `fzf` — fuzzy finder для истории команд

```bash
# Установка oh-my-zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

## Редактор кода

VSCode с минимальным набором расширений:

1. Prettier — форматирование кода
2. ESLint — линтинг JavaScript
3. GitLens — продвинутая работа с Git
4. Material Icon Theme — красивые иконки файлов

## Продуктивность

Несколько правил, которые помогают:

- **Pomodoro** — 25 минут работы, 5 минут отдых
- **GTD** — всё выписывать, ничего не держать в голове
- **Single-tasking** — одна задача за раз
